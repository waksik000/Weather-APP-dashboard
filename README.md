# 🌤️ Smart Weather Dashboard

Учебный React-проект для прокачки:
- useState
- useEffect
- useRef
- useMemo
- работа с API
- работа с localStorage
- debounce логика

---

# 🎯 Цель проекта

Создать погодное приложение, которое позволяет:
- искать погоду по городам
- сохранять историю запросов
- сравнивать погоду разных городов
- оптимизировать API-запросы (debounce)
- работать с локальным хранилищем

---

# 🧠 Основной функционал

## 1. Поиск погоды по городу

### Функционал:
- пользователь вводит название города
- приложение делает запрос к API погоды
- отображает текущую погоду

### Требования:
- поиск через input
- кнопка "Search" (или авто-запрос через debounce)
- обработка ошибок (не найден город, нет интернета)

---

## 2. Отображение погоды

Отображаем:

- температура (°C)
- ощущается как
- влажность
- скорость ветра
- описание погоды (clear, cloudy и т.д.)
- иконка погоды (если доступна из API)

---

## 3. Loading / Error состояния

Приложение должно иметь 3 состояния:

- `loading` → показывается спиннер или текст
- `error` → сообщение об ошибке
- `success` → отображение данных

---

## 4. История поисков

### Функционал:
- сохранять последние 5 городов
- при клике по городу → повторный запрос
- сохранять данные в `localStorage`

### Требования:
- при обновлении страницы история сохраняется
- новые города добавляются в начало списка
- максимум 5 элементов

---

## 5. Favorites (избранное)

### Функционал:
- добавить город в избранное (⭐)
- удалить из избранного
- хранение в `localStorage`

---

## 6. Debounce (защита от лишних запросов)

### Логика:
- запрос к API НЕ должен выполняться на каждый ввод символа
- запрос выполняется через 500–800ms после остановки ввода

### Требования:
- использовать `useRef` для хранения таймера
- очищать предыдущий таймер перед новым

---

## 7. Compare Mode (сравнение городов)

### Функционал:
- пользователь может выбрать 2 города
- отображается сравнение:

  - температура
  - влажность
  - ветер

### Логика:
- использовать `useMemo` для вычисления результата сравнения

---

# ⚙️ Используемые технологии и хуки

## React хуки:
- useState → управление состоянием UI
- useEffect → API запросы и localStorage sync
- useRef → debounce таймеры и DOM доступ
- useMemo → оптимизация сравнения данных
